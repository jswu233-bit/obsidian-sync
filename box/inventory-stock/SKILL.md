# 囤货库存系统

## 使用方式

### 1. 增加囤货（入库）

告诉我要买什么，我记录入库：
```
入库 [物品名] [数量]
```

示例：
- `入库 心相印抽纸 2提`
- `入库 棉签 3包`

### 2. 更新库存（盘点）

拍照或告诉我剩余数量：
```
盘点 [物品名] 剩 [数量]
```

示例：
- `盘点 抽纸 剩 8包`
- `盘点 口罩 剩 3包`

### 3. 记录消耗（可选）

如果你想主动记录用了多少：
```
消耗 [物品名] [数量]
```

### 4. 查询

- `查 抽纸` - 查某物品库存
- `库存` - 显示全部库存
- `周报` - 手动生成周报

### 5. 设置消耗速度

运行一段时间后，告诉我：
```
设置 [物品名] 每月消耗 [数量]
```

示例：
- `设置 抽纸 每月消耗 6包`

## 自动功能

**每周日晚8点**，我会自动发送：
- 当前库存清单
- 预计还能用多久
- 建议购买数量

## 数据文件

- `inventory/stock.yaml` - 库存数据
- `inventory/囤货清单.md` - 可视化清单
