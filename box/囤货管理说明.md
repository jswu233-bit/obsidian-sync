# Inventory Manager - 囤货管理器

## 功能

- 按分类记录囤货物品
- 支持入库日期追踪
- 库存统计汇总
- Markdown 表格格式，易读易编辑

## 分类体系

1. **纸类** - 抽纸、卷纸、湿巾、酒精湿巾、小包纸巾
2. **厨房用品** - 除尘掸、厨房纸、保鲜膜等
3. **清洁用品** - 吸色片、防霉剂、湿厕纸、清洁剂等
4. **旅行用品** - 一次性浴巾、一次性内裤、旅行装等
5. **个人护理用品** - 牙线、棉签、口罩、护肤品等

## 使用方式

### 添加新物品

发送消息格式：
```
入库 [分类] [品牌] [物品名] [数量] [规格(可选)] [备注(可选)]
```

示例：
- `入库 纸类 维达 抽纸 3包 超韧系列`
- `入库 清洁用品 - 防霉剂 2包 放衣柜用`

### 查看库存

- `查看囤货清单` - 显示完整清单
- `查看纸类` - 显示指定分类

### 更新库存

- `消耗 [分类] [物品名] [数量]` - 减少库存
- `修改 [分类] [物品名] [字段] [新值]` - 修改记录

## 文件位置

- 清单文件：`inventory/囤货清单.md`
- 备份位置：`obsidian-sync/box/囤货清单.md`

## 字段说明

| 字段 | 说明 |
|------|------|
| 品牌 | 商品品牌，无品牌填 `-` |
| 物品 | 物品名称 |
| 规格 | 详细规格描述 |
| 数量 | 当前库存数量 |
| 入库日期 | 格式：YYYY-MM-DD，自动记录 |
| 备注 | 其他补充信息 |
