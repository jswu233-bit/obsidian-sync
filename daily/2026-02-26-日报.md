**1. Anthropic: Claude Code 改变了编程，Claude Cowork 要改变所有知识工作** 🔥
💬 Anthropic 在"Enterprise Agents"发布会上承认，2025年企业AI Agent的试点"大多数是早产儿"——不是努力不够，是方法不对。他们的新答案是 Claude Cowork：把 Claude Code 在开发者圈子验证过的"委托式工作"模式，复制到HR、设计、金融分析、投行等所有知识工作场景。
📌 关键细节：
• 企业管理员可以搭建私有插件市场，连接内部 GitHub 仓库作为插件源
• 新增 MCP 连接器，打通 Google Drive、Gmail、DocuSign、FactSet
• 预置了投行、PE、财富管理等垂直行业的插件模板
• 企业业务已占 Anthropic 收入的约 80%
💡 Zoe 点评：这不是"AI助手"了，这是"AI同事"。Anthropic 从 API 公司转型 toB 产品公司的信号非常明确，对做 AIGC 工具的产品经理来说，这个方向值得深挖。
📎 [VentureBeat](https://venturebeat.com/orchestration/anthropic-says-claude-code-transformed-programming-now-claude-cowork-is) | [CNBC](https://www.cnbc.com/2026/02/24/anthropic-claude-cowork-office-worker.html)

**2. Nvidia Vera Rubin 首次曝光：能效是 Blackwell 的 10 倍** 🚀
💬 CNBC 独家探访了 Nvidia 圣克拉拉总部，首次公开下一代 AI 系统 Vera Rubin 的实物。这套系统由 130 万个零部件组成，核心是 72 颗 Rubin GPU + 36 颗 Vera CPU，来自 20+ 国家的 80+ 供应商。
📌 关键数据：

## 🐦 今日 AI 圈热门观点（替代 X 博主板块）

> ⚠️ X 搜索 cookies 已过期，今日改用全网热门观点替代。已记录修复任务。

**观点 1：「每家公司的个体决策都是理性的，集体结果是灾难性的」**
— James Van Geelen (Citrini Research CEO)
💬 这句话出自那篇 2300 万阅读的"AI末日备忘录"，精准概括了 AI 替代劳动力的囚徒困境：不裁员就被竞争对手干掉，全裁员就没人消费了。
🔍 为什么关注：这不是反 AI 的论调，而是在提醒所有做 AI 产品的人——你的产品在宏观层面会产生什么二阶效应？

**观点 2：「2025年企业AI Agent试点大多是早产儿，不是努力不够，是方法不对」**
— Kate Jensen (Anthropic 美洲区负责人)
💬 Anthropic 在发布会上直接承认去年的 Agent 热潮"大部分没跑通"。他们的反思是：不应该让 Agent 从零开始做事，而应该像 Claude Code 那样，让 AI 接手人类已经在做的工作流。
🔍 为什么关注：这是大厂第一次公开承认 Agent 落地的困境，同时给出了"从工具到同事"的新路径。对产品经理来说，这个思路转变很关键。

**观点 3：「AI Agent 使推理 token 消耗提升 4-15 倍」**
— 财闻网行业分析
💬 Agent 不是"调一次 API"就完事，它需要反复推理、规划、执行、验证。这意味着算力消耗是传统 chatbot 的 4-15 倍，多模态场景更是 5-10 倍。
🔍 为什么关注：做 Agent 产品的成本结构和 chatbot 完全不同，定价模型需要重新设计。
