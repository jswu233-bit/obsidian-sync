# 日报生成技能（A版）

## 目标
产出“精炼且详细”的日报：每条必须包含
1) 发生了什么
2) 为什么重要
3) Zoe点评

## 角色分工
- Spy：负责搜集原始信息
- Zoe：负责判断、提炼、点评、成稿

## 强制协作纪律（Leader规则）
- Spy 若反馈“受限/找不到”，不得结束任务。
- Zoe 必须继续施压并要求 Spy 立刻切换备用工具链补齐。
- 备用顺序：
  1. 原工具
  2. web_search（site定向）
  3. web_fetch（抓正文）
  4. 可信二手源交叉验证
- 最终稿禁止出现：未获取 / 留空 / 待补 / xxxxx。

## 输出模板（A版）
- 标题：`# 📰 Jamie每日综合日报 - YYYY年M月D日（Update版）`
- 顶部含：更新时间 + 版本说明
- 板块顺序：
  1. 🤖 AI新闻板块
  2. 🐦 X博主动态（完整解读版）
  3. 📱 微信公众号精选
  4. 📈 基金与金融市场
  5. 🌤️ 北京天气
  6. 📌 日报总结（今日热点 + 今日洞察）

## 发布规则
- 文件命名：`daily/YYYY-MM-DD-update*.md`
- Git 分支：`main`
- 提交后回报 commit hash
