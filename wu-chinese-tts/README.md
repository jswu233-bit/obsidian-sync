# 🎭 文字转吴语 - 吴语拼音转换器

一个优雅的文字转吴语拼音网站，支持上海话、苏州话、绍兴话等多种吴语方言。

## ✨ 功能特点

- 📝 **文字转拼音**：输入中文，自动转换为吴语拼音
- 🎙️ **语音朗读**：支持女声/男声朗读（使用浏览器TTS）
- 🌐 **多方言支持**：上海话、苏州话、绍兴话
- 📱 **响应式设计**：支持手机、平板、电脑访问
- ⌨️ **键盘快捷键**：Ctrl+Enter 快速转换，Esc 停止播放

## 🚀 快速启动

### 前置要求
需要先安装 `edge-tts` Python 包：
```bash
pip install edge-tts
```

### 启动服务器
```bash
cd wu-chinese-tts
python3 server.py
```

然后访问 `http://localhost:8080`

### 功能说明
- 🎙️ **Edge-TTS 语音合成**：使用微软 Edge 的在线语音服务
- 👩 支持多种女声：晓晓（活泼）、晓伊（温柔）
- 👨 支持多种男声：云健、云希（年轻）
- 🔊 播放速度已调整为 0.9 倍，更接近方言节奏

## 🔧 技术说明

### 拼音数据来源
- 上海话：基于[吴语学堂](https://www.wugniu.com/)拼音方案
- 苏州话：基于[吴语协会](http://wu-chinese.com/)拼音方案
- 绍兴话：基于《绍兴市志》及《绍兴方言同音字汇》

### 语音合成
- 当前使用浏览器内置 Web Speech API
- 支持中文普通话朗读（吴语TTS资源较少）
- 可扩展接入 Azure TTS、阿里云TTS等服务

## 📝 使用示例

输入文字：
```
你好，欢迎来到上海！
```

转换结果：
```
你[non] 好[hau] ， 欢[hue] 迎[gnin] 来[le] 到[tau] 上[zaon] 海[he] ！
```
