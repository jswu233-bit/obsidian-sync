# 🎭 文字转吴语 - 吴语拼音转换器

一个简单易用的网页工具，可以将汉字转换为吴语拼音（罗马字），并提供语音朗读功能。

## ✨ 功能特点

- 🔤 **多方言支持**: 上海话、苏州话、绍兴话
- 📝 **实时转换**: 输入文字即时显示吴语拼音
- 🔊 **语音朗读**: 支持女声朗读（使用浏览器 Web Speech API）
- 🎨 **精美界面**: 现代化的UI设计，支持移动端
- 📱 **响应式**: 完美适配手机、平板、电脑

## 🚀 快速开始

### 方式一：直接打开
直接在浏览器中打开 `index.html` 文件即可使用。

### 方式二：使用本地服务器
```bash
# 使用 Node.js
node server.js

# 或使用 Python 3
python -m http.server 3000

# 或使用 Python 2
python -m SimpleHTTPServer 3000
```

然后访问 http://localhost:3000

## 📖 使用说明

1. **选择方言**: 点击顶部的方言按钮选择要转换的吴语方言（上海话/苏州话/绍兴话）
2. **输入文字**: 在输入框中输入要转换的汉字
3. **点击转换**: 点击"转换为吴语拼音"按钮
4. **查看结果**: 下方会显示每个汉字对应的吴语拼音
5. **朗读功能**: 点击"女声朗读"按钮可以听到发音

### 快捷键
- `Ctrl + Enter`: 快速转换

## 📝 拼音方案说明

本工具使用的拼音方案来自 [吴语学堂](https://www.wugniu.com/) 和 [吴语协会](http://wu-chinese.com/)，是目前最权威的吴语罗马字方案。

### 示例

| 汉字 | 上海话 | 苏州话 | 绍兴话 |
|------|--------|--------|--------|
| 我 | ngu | ngu | ngo |
| 你 | non | ne | non |
| 好 | hau | hau | hau |
| 谢谢 | xia-xia | zia-zia | xia-xia |

## 🔊 关于语音朗读

目前使用浏览器的 Web Speech API 进行朗读，支持中文普通话语音。

**注意**: 由于吴语TTS技术限制，目前朗读使用的是普通话发音。如需真实的吴语发音，可以考虑接入：
- 阿里云语音合成（支持部分方言）
- 腾讯云语音合成
- 科大讯飞方言合成
