# 📁 文件夹结构说明

这个仓库用于 VPS (OpenClaw) 和本地 Obsidian 之间的双向同步。

## 目录结构

```
obsidian-sync/
├── 📥 inbox/           # Zoe 发送的信息（查询结果、资讯等）
├── 📤 outbox/          # Zoe 需要 Jamie 确认/处理的信息
├── 📚 knowledge/       # 整理后的知识库（双方共同维护）
└── 📝 daily/           # 每日日志（Zoe 维护）
```

## 使用说明

| 文件夹 | 用途 | 谁写入 |
|--------|------|--------|
| **inbox/** | Zoe 发送的查询结果、资讯、文章等 | Zoe |
| **outbox/** | Zoe 需要 Jamie 确认、回复或处理的事项 | Zoe |
| **knowledge/** | 整理后的知识库，可长期查阅 | 双方 |
| **daily/** | 每日工作日志和总结 | Zoe |

## 工作流程

1. **Zoe 查询内容** → 写入 `inbox/`
2. **Zoe 需要确认** → 写入 `outbox/`
3. **Git push** → 同步到 GitHub
4. **Jamie 本地 pull** → 在 Obsidian 中查看
5. **Jamie 整理后** → 移动到 `knowledge/`
6. **Git push** → Zoe 可以看到更新
